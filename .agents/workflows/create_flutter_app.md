# Flutter App Creation Workflow

This workflow sets up a new Flutter application and runs it.

1. **Create the Flutter project**: run `flutter create <project_name>` (replace `<project_name>` with your actual project name).
2. **Navigate to the directory**: run `cd <project_name>`.
3. **Run the app**: run `flutter run` to launch the application.
